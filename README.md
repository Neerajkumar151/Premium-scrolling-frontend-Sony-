## âœ¦ Getting Started

Experience the silence on your local machine.

### 1. Clone the Experience

```bash
git clone https://github.com/your-username/sony-webapp.git
cd sony-webapp
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Ignite functionality

```bash
npm run dev
```

Visit `http://localhost:5173` to view the application.
